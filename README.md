## Instalación

Primero instala las dependencias necesarias.

```bash
npm install baileys
```

Luego instala el paquete para mostrar el QR en la terminal:

```bash
npm install qrcode-terminal
```

Luego ejecuta

```bash
node index.js
```
El archivo index.js es solo para probar que se envía el mensaje

El archivo sendMessages.js es el que envía los mensajes a un array de números, solo lo probé con mi número y si enviaba, le tengo fe a que envía normal mínimo a 26, con fe
